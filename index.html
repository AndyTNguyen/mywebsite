What is a responsive Website?


A responsive website is a website that can adjust itself accordingly to fit the user's screen resolution and to provider a better user experience. For an example, Phone users have a smaller screen and having a responsive design can be very beneficial. The content, line of text, images, etc can fit on the screen without getting cut off if the website has responsive attributes. To test the responsiveness of my website, resize the internet browser. Notice how the image automatically resizes itself without getting cut off, it dosn't becoming disproportionate and it dosn't become pixelated. Same rule applies to my paragraphs and text. Thats because my code made the picture respond to the current side of your screen. You can explore the rest of my projects by clicking the menu bar at the top left corner.
